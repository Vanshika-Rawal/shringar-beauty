SHRINGAR — Brand logo files
============================

Drop each brand's REAL logo image in THIS folder (public/brands/) using the
exact filename below. Supported formats (tried in this order): .png, .svg,
.webp, .jpg  →  e.g. "loreal-paris.png" or "mac.svg".

Tips for a clean Nykaa-style grid:
- Use PNG or SVG with a TRANSPARENT background.
- Trim extra whitespace; keep the logo roughly centered.
- Any size is fine (they scale to fit); ~400px wide looks crisp.

Until a file is added, the grid shows an elegant text wordmark automatically —
so it always looks complete. Add files anytime and they appear instantly.
