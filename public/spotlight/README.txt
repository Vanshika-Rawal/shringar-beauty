SHRINGAR — Spotlight house images
==================================

These are the hero shots for the "Houses we're loving" cards on /brands.

Drop ONE image per house in THIS folder (public/spotlight/) using the exact
filename below. Supported: .jpg, .png, .webp, .svg — e.g. "dove.jpg".

The cards are portrait (3:4), so a tall/square product shot looks best.
Anything roughly 800x1000px or larger will be crisp.

Until a file is added, that card shows an elegant gradient with the house
name — the row never looks broken. Add files anytime; they appear instantly.

Filename for each house currently in the spotlight:
--------------------------------------------------
Vaseline     ->  vaseline.jpg
Dove         ->  dove.jpg
Dot & Key    ->  dot-and-key.jpg
CeraVe       ->  cerave.jpg
Plum         ->  plum.jpg
Dior         ->  dior.jpg

(To change which houses appear, edit SPOTLIGHT in app/brands/page.tsx —
 the filename is just the house name lowercased, spaces/& turned into "-and-".)
