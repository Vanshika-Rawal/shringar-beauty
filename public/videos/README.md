# SHRINGAR — luxury videos

Drop `.mp4` clips here and they auto-play in the UI. Until a file exists, each
slot gracefully shows an animated rose-gold gradient scene, so the site already
looks premium without any assets.

## Shop by Category tiles (`components/home/CategoryGrid.tsx`)

- `category-1.mp4` … `category-14.mp4` (plays on hover, in category order:
  Skincare, Makeup, Hair Care, Body Care, Fragrance, Bath & Shower, Lip Care,
  Eye Care, Korean Beauty, Luxury Beauty, Men's Grooming, Gift Sets, Beauty
  Tools, Wellness)

Recommended: 600×720 portrait, 4–6s loop, muted, < 2 MB each.

## Build Your Routine cards (`components/home/DiscoverySections.tsx`)

- `routine-1.mp4` … `routine-5.mp4` (plays on hover)
